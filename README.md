# JavaScript Fundamentals & DOM Practice

This repository contains practical exercises focused on core JavaScript concepts and browser-based development.

## Covered Topics

- JavaScript fundamentals
- Arrays and algorithms
- Object-oriented programming (classes)
- DOM manipulation
- Event handling
- Dynamic DOM updates
- Forms and table interactions
- Drag & keyboard interactions
- Browser APIs
- Local Storage
- Timers and intervals

## Purpose

This project demonstrates foundational frontend development skills using vanilla JavaScript, focusing on interactive and dynamic web applications.
